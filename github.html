About This Site

This site was hand-crafted just to demonstrate my favorite GitHub-flavored technologies. It was lovingly built using Atom, the GitHub repo is live on my personal account, and I'm hosting this Jekyll-powered static site on GitHub pages.

The design process followed my normal progression from rapid sketch iterations to digital mockups before diving into code.

For this project, I used the beta release Adobe Xd for the digital mockups and early prototyping. I've fallen in love with this workflow and look forward to using it much more.

In the code, I use SASS and a custom Gulp.JS setup that I modified from ShakyShane's popular repo so I could use my favorite development tools. I followed a fairly strict BEM methodology in the CSS to further explore object-oriented styling. The resulting CSS Stats show the value of this approach in keeping a very flat specifity graph and a small filesize.
