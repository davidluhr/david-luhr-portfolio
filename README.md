# Luhr.co

## Setup

```bash
git clone git@github.com:davidluhr/david-luhr-portfolio.git
```

```bash
cd david-luhr-portfolio
```

```bash
npm install
```

## Local development

```bash
npm run serve
```

Open the local development preview at [localhost:8080](http://localhost:8080).


## Built with

### Design
- [Figma](https://figma.com/)

### Code
